# Эксперимент: Vision Catalog Describer

**Дата:** 2026-04-24
**Цель:** Подобрать промпт Claude Vision который даёт стабильный structured JSON для последующего поиска по каталогу MoySklad.

## Что делаем

Через Flowise UI (`http://localhost:3130`) создаём chatflow, грузим туда фотки фильтров / картриджей / edge-cases (кот, документ), смотрим что возвращает Claude Sonnet. Итерируем промпт до стабильного output.

## Структура папки

- `prompt-v1.md` — текущий промпт (system + examples + тест-чеклист)
- `prompt-v2.md` / `v3.md` — если v1 показал проблемы, пишем новую версию
- `results/<prompt-version>/<image-name>.json` — JSON outputs для каждой фото
- `results/<prompt-version>/report.md` — сводка "сколько из N тестов прошло"

## Тестовые фото

Собрать 10-15 штук:

- [ ] 3-4 реальных фильтра Аквафор (разные модели) — happy path
- [ ] 1 картридж вблизи с надписями — model_hint check
- [ ] 1 коробка/упаковка — распознавание по дизайну
- [ ] 1 размытое/тёмное фото — confidence=low check
- [ ] 1 бланк анализа воды — is_relevant=false (не оборудование)
- [ ] 1 кот или случайная бытовуха — is_relevant=false
- [ ] 1 фильтр конкурента (Гейзер/Atoll) — brand recognition
